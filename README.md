# 🎸 핫한 공연 예매의 시작, 불티
> 불티(Boolti)는 다양한 공연을 손쉽게 예매하고 입장할 수 있게 도와주는 플랫폼이에요.

> 공연을 즐기고 싶은 모두가 더 쉽게 공연을 주최하고 관람할 수 있도록 **'불티'** 에서 다양한 공연을 확인해보세요!
